# Roadmap sugerido

## Fase 0.1 - Vertical slice jugable

Objetivo: una canción completa, jugable de inicio a fin.

- Input de 5 carriles.
- Notas normales + sustain.
- Score/combo/multiplicador.
- Wwise básico (music + hit/miss + UI).
- Pantalla de resultados.

Criterio de salida:
- Build interna estable en PC.
- Sin desincronización percibida grave.

## Fase 0.2 - Solidez de producto

- Dificultades (Easy/Normal/Hard).
- 3-5 canciones.
- Calibración audio/input in-game.
- Mejoras visuales de feedback.
- Persistencia local de highscores.

Criterio de salida:
- Sesiones largas sin degradación de performance.
- Telemetría básica de errores.

## Fase 0.3 - Escalado y contenido

- Pipeline de ingestión de charts semiautomática.
- Sistema de mods internos (si aplica).
- Ajustes avanzados de accesibilidad.
- Packaging de contenido por DLC/canción.

Criterio de salida:
- Flujo reproducible para agregar una canción nueva en < 1 hora.
